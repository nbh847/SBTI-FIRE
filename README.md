# 🔥 FIRE-SBTI 财务自由人格测试

MBTI已经过时，FIRE-SBTI来了。

## 简介

这是一个基于**15个财务维度**的娱乐测试，将你分类为**27种财务人格类型**之一。你是FIRE达人？月光族？还是币圈HODL？测试仅需2分钟，让你的财务人格无处遁形。

## 特性

- 🧠 **15个财务维度**：财富、风险、消费、规划、投资五大模型
- 🎭 **27种搞怪人格**：FIRE达人、月光族、赌狗、币圈HODL等
- ⚡ **纯前端运行**：无需后端，打开即用
- 📱 **响应式设计**：支持手机端访问

## 15个财务维度

| 模型 | 维度 |
|------|------|
| 财富模型 | 资产规模认知、收入结构多元化、净资产意识 |
| 风险模型 | 风险承受力、恐慌指数、杠杆态度 |
| 消费模型 | 消费冲动度、极简倾向、社交消费压力 |
| 规划模型 | FIRE目标清晰度、储蓄率稳定性、退休紧迫感 |
| 投资模型 | 投资知识水平、主动/被动偏好、收益预期 |

## 部分人格类型

- 🔥 **FIRE达人** - 储蓄率50%+，10年内退休
- 🏖️ **躺平FIRE** - 已有基础，轻松工作
- ☕ **兼职FIRE** - 退休后打零工
- 🥗 **苦行僧FIRE** - 极简生活，低成本退休
- 🥩 **富贵FIRE** - 高消费FIRE
- 🐹 **仓鼠党** - 只存不花，放银行不敢投
- 🌙 **月光族** - 月光，有多少花多少
- 🎲 **赌狗** - 追求暴富，高杠杆合约
- 💎 **币圈HODL** - 全仓山寨币，拿住
- 📈 **指数信徒** - Boglehead，定投标普500
- ...更多等你发现
